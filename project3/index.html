Rules

Welcome to the Rules page of the Picture Perfect Photography online photo club! This will help guide you to know about what is allowed and not allowed within the web pages. There are guidelines when it comes to accessing this site, and you must follow it so that everything can go smoothly and well. For the first rule, you must use a specific file format for ALL images. This means that when you send or upload an image, it must be in JPEG, JPG, PNG, or GIF format. No other format will be accepted or taken. For the second rule, all image file sizes must not exceed more than 1MB. Any image file over 1MB will be deleted and not taken into consideration, so please beware and watch your image file size.

For the third rule, you must be able to post a picture that is in accordance with the themes and categories of the online photography club. The categories are mentioned in the category navigation link that is located within the main front homepage of this site. Make sure you know which categories are allowed, which are the ones mentioned within the category page, and if you upload or submit any other picture that is not related to the categories mentioned within the categories page, the picture will immediately not be taken into consideration and will be deleted. For the fourth rule, all of the pictures that you upload must be original. This means that the picture relating to the category that you are posting must be your own original picture to submit and upload. That is all for the rules! Enjoy taking pictures relating to the categories, and don’t forget to submit them soon!
